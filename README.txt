F1 Website Copy using HTML:
This repository contains a basic copy of the Formula 1 (F1) website, recreated using only HTML.
This project aims to showcase my HTML skills and replicate the structure and layout of the F1 website. 
Please note that this is a static version and doesn't include any interactivity or dynamic features.
-INTRODUCTION:
The Formula 1 (F1) website is a prominent platform for motorsport enthusiasts, providing up-to-date information about races, teams, drivers, news, and more.
This project is a static replica of the F1 website, built entirely with HTML. It demonstrates my ability to create a visually appealing and well-structured website using the basics of HTML.
-FEATURES:
Main page of the f1 site
Some news and featurings
A sign-in direct link
-INSTALLATION:
Clone this repository to your local machine using your preferred method
Navigate to the project's root directory.
